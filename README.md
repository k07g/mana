# mana

対話型アシスタント CLI

## 必要環境

- Go 1.23 以上

## インストール

```bash
go install github.com/k07g/mana@latest
```

または、リポジトリをクローンしてからビルドする場合:

```bash
make install
```

いずれも `$(GOPATH)/bin` にバイナリがインストールされます。

## アンインストール

```bash
make uninstall
```

## 使い方

```bash
mana
```

### オプション

| オプション | 説明 |
| --- | --- |
| `-v`, `--version` | バージョンを表示して終了する |

### 対話コマンド

起動後、プロンプト `>` に対して以下の入力に反応します。

| 入力例 | 動作 |
| --- | --- |
| `おはよう` / `こんにちは` など | 挨拶を返す |
| `バージョン` を含む文 | アプリのバージョンを表示する |
| `今日の天気` を含む文 | 現在地の天気と気温を表示する |
| `quit` / `exit` / `さようなら` など | 終了する |

## 開発

```bash
# 実行
make run

# テスト
make test

# ビルド
make build

# ビルド成果物を削除
make clean
```
